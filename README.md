# 🚌 Smart Bus Tracking System

A real-time Smart Bus Tracking System that allows users to track bus locations, view estimated arrival times, and explore routes efficiently using map integration.

## 📌 Features
- Real-time tracking
- ETA prediction
- Route details
- Map integration

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- MongoDB

## 🚀 Getting Started
git clone https://github.com/preetisahu25/Smart-Bus-locator--Bus-tracking-system.git
cd Smart-Bus-locator--Bus-tracking-system
npm install
npm start
